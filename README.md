# Commands
Do @help in a server with random.bot, or DM it.
Some commands include: 
@someone,@iamveryrandom(randomly kicks / bans a user), @eightball, @rnd, @coinflip, @choose, @shuffle, @name, @team, @suggest.
Do note that the bot is still under development.
